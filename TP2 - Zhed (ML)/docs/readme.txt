Para correr o programa:

1. Instalar o plugin ML-agents com Unity
2. Criar um projeto a partir da pasta ZhedUnity (ou ZhedUnityRegular)
3. Carregar em Play
4. Escolher o nível inicial

Os níveis vão rodando aleatoriamente, ficando o agente verde quando ganha, vermelho quando perde.

Nota: 
- Botão graphics altera as animações do tabuleiro para ficarem mais apelativa. (Também é a única maneira de jogar como utilizador)
- Botão restart desliga os agentes, passando a ser possível jogar como utilizador.
